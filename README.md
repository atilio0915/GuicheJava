## Guiche java
Neste projeto, eu tentei representar a logica de um guiche de atendimento da UAI-BH, colocando em pratica algumas estruturas do Java que aprendi no curso da Alura.

# Tecnologias usadas 
-java

## Requisitos
- Java JDK 8 ou superior
- (Opcional) IntelliJ IDEA ou outro IDE Java

## Rodando pelo terminal (Windows)
1) Abra o PowerShell na pasta do projeto.
2) Compile os arquivos:
   ```powershell
   javac src\*.java
   ```
3) Execute o programa:
   ```powershell
   java -cp src Main
   ```

## Rodando pelo IntelliJ IDEA
1) Abra o projeto no IntelliJ.
2) Abra o arquivo `src/Main.java`.
3) Clique no botao de Run (ou use Shift + F10).

## Como usar o programa
Ao iniciar, escolha uma opcao do menu:
0 - criar guiche
1 - criar cliente
2 - atender cliente
3 - cliente atendido
4 - sair
